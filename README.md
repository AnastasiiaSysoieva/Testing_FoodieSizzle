# Testing_FoodieSizzle

This repository contains QA testing documentation for the mobile cooking game **Foodie Sizzle**.

## Project Overview

Foodie Sizzle is a mobile cooking game where players prepare vegetable dishes for customers using different grills.  
The game includes a tutorial, multiple levels, and special grill mechanics.

## Testing Scope

The following features were tested:

- Tutorial system
- Gameplay mechanics
- Grill types
- Order processing
- Level progression
- Life system
- Game resume after interruption

## Testing Types

The project includes:

- Functional testing
- UI testing
- Exploratory testing
- Regression testing
