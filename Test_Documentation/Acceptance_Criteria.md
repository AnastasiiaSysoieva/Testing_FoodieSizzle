# Acceptance Criteria

## AC-01 Daily Reward
- Reward appears once per day
- Player can collect coins
- Coins are added to balance

## AC-02 Start Level
- Level loads successfully
- Player can interact with objects

## AC-03 Customer Orders
- Customer order is visible
- Player can serve the order

## AC-04 Vegetable Combination
- Vegetables can be combined
- Correct dish is created

## AC-05 Special Grill
- Vegetables placed on red grill become spicy

## AC-06 Timer
- Timer appears on screen
- Timer counts down correctly

## AC-07 Difficulty
- Level difficulty varies randomly

## AC-08 Life System
- Player loses one life after failing

## AC-09 Resume Game
- Game resumes after reopening app

## AC-10 Unlock Grill
- New grill appears after completing levels

## AC-11 Shop
- Player can purchase items

## AC-12 Tutorial
- Tutorial starts on first launch
- Only required actions allowed
