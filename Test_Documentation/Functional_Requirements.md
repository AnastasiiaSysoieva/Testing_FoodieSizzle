# Functional Requirements

## Feature: Tutorial System

FR-1: The game provides a tutorial on the first level for new players.

FR-2: Tutorial introduces basic gameplay mechanics.

FR-3: Tutorial actions guide the player step-by-step.

## Feature: Grill Mechanics

FR-4: Food items are placed on grills.

FR-5: Player can combine food items according to game rules.

FR-6: New grill types unlock as player progresses through levels.

FR-7: Red grills modify food by making items spicy.

## Feature: Level Gameplay

FR-8: Player starts a level using one life.

FR-9: Each level has a countdown timer.

FR-10: Customers appear and request food orders.

FR-11: Orders must be completed within the time limit.

## Feature: Reward System

FR-12: Player receives daily login rewards.

FR-13: Coins are added to balance after rewards.

## Feature: Life System

FR-14: Player loses a life when exiting a level.

FR-15: Player cannot start a level without available lives.
