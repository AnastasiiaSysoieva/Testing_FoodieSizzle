# Use Cases

## UC-01 Start Level

Actor: Player

Preconditions:
- The player has at least one life.
- The game is launched.

Main Flow:

1. Player opens the game.
2. Player presses "Start Level".
3. The level loads.
4. The gameplay begins.

Postconditions:

- The level starts successfully.

---

## UC-02 Serve Customer

Actor: Player

Preconditions:

- A customer appears with an order.

Main Flow:

1. Player collects the required vegetables.
2. Player prepares them on the grill.
3. Player serves the dish to the customer.

Postconditions:

- Customer order is completed.

---

## UC-03 Receive Daily Reward

Actor: Player

Preconditions:

- Player logs in to the game.

Main Flow:

1. Player opens the game.
2. The system displays the daily reward screen.
3. Player collects coins.

Postconditions:

- Coins are added to the player's balance.
