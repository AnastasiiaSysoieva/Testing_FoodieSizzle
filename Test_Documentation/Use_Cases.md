# Use Cases

UC-01 Receive Daily Reward

Actor: Player

Main Flow
1. Player opens the game
2. System shows daily reward screen
3. Player collects reward

---

UC-02 Start Level

Actor: Player

Main Flow
1. Player presses Start Level
2. System loads the level
3. Gameplay begins

---

UC-03 Serve Customer

Actor: Player

Main Flow
1. Customer appears with order
2. Player prepares vegetables
3. Player serves the order

---

UC-04 Combine Vegetables

Actor: Player

Main Flow
1. Vegetables appear on grills
2. Player combines vegetables
3. Dish is created

---

UC-05 Use Special Grill

Actor: Player

Main Flow
1. Player places vegetables on red grill
2. Vegetables become spicy

---

UC-06 Level Timer

Actor: Player

Main Flow
