# User Stories

### US-01 Daily Reward

As a player  
I want to receive a daily reward when I log into the game  
So that I feel motivated to return to the game every day.

---

### US-02 Start Level

As a player  
I want to start a new level  
So that I can progress through the game.

---

### US-03 Complete Customer Orders

As a player  
I want to prepare vegetables and serve them to customers  
So that I can complete the level objectives.

---

### US-04 Use Different Grills

As a player  
I want to cook vegetables on different types of grills  
So that gameplay becomes more interesting.

---

### US-05 Resume Interrupted Game

As a player  
I want to resume my game if the application is interrupted  
So that I don't lose my progress.
