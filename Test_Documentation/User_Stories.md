# User Stories

## US-01  
As a player  
I want to receive a daily reward  
So that I am motivated to return to the game every day.

## US-02  
As a player  
I want to start a level  
So that I can progress through the game.

## US-03  
As a player  
I want to complete customer orders  
So that I can earn points.

## US-04  
As a player  
I want vegetables to appear on grills  
So that I can combine them into dishes.

## US-05  
As a player  
I want different types of grills  
So that gameplay becomes more diverse.

## US-06  
As a player  
I want to see a timer during the level  
So that I know how much time remains.

## US-07  
As a player  
I want the level difficulty to vary  
So that the game remains interesting.

## US-08  
As a player  
I want to lose a life when I fail a level  
So that the game has consequences.

## US-09  
As a player  
I want the game to resume after interruption  
So that I don't lose my progress.

## US-10  
As a player  
I want to unlock new grills as I progress  
So that new mechanics appear.

## US-11  
As a player  
I want to buy items or upgrades  
So that I can improve gameplay.

## US-12  
As a player  
I want a tutorial at the beginning  
So that I understand how to play the game.
